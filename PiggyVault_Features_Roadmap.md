# PiggyVault Savings — Дорожня карта фіч

> Гейміфікований фінансовий додаток для заощаджень | v2.0

---

## 1. Соціальна складова (найважливіше)

### 1.1 Друзі / Команда мрії
- Запрошування друзів та відстеження їхнього прогресу
- Анонімний режим або з дозволу користувача
- Спільний дашборд прогресу команди

### 1.2 Спільні цілі (Альянс 2.0)
- Створення групових фінансових цілей (напр. "на PlayStation для тусовки")
- Візуальний прогрес кожного учасника
- Автоматичний розподіл внесків

### 1.3 Лідерборди
- Тижневі / місячні таблиці найкращих "зберігачів"
- Анімація підйому в рейтингу
- Категорії: індивідуальні, командні, альянси

### 1.4 Мікро-донати
- Можливість "підкидати" один одному маленькі суми
- Донат у спільну ціль
- Нотифікація + реакція на отриманий донат

### 1.5 Спільні челенджі
- Створення групових цілей (напр. "Зберегти 10 000 грн разом за місяць")
- Ранги: Бронза → Срібло → Золото → Платина → Діамант
- Унікальні привілеї для кожної ліги

## 2. Гейміфікація та прогресія

### 2.1 Дерево навичок (SP)
- **Гілка "Фінансова дисципліна"** — щоденні внески, стабільність, регулярність
- **Гілка "Інвестиції"** — мікро-інвестиції, ETF, розумні цілі
- **Гілка "Імпульс-контроль"** — контроль витрат, анти-штрафи, стрік-захист
- Unlockable навички за SP (skill points)

### 2.2 Рівні (Level 1 → Level 50)
- Візуальні покращення аватара кібер-робота
- Нові unlocked-елементи на кожному рівні
- Boost-бонуси за рівнем (більше XP, кращі нагороди)

### 2.3 Достиження (Achievements)
- Бейджі та анімовані трофеї
- Unlockable аватари за виконані досягнення
- Рідкісні / легендарні нагороди

### 2.4 Щоденний бонус-календар
- Щоденний вхід → бонус, що зростає зі стріком
- День 1: +5 кристалів, День 7: +50 кристалів
- "Золотий день" раз на місяць — подвійний бонус

### 2.5 Сезонні події
- "Чорна п'ятниця Challenge" — анти-імпульсні квести
- "Новий рік 2027" — фінансові резолюції
- "Літній марафон заощаджень" — тематичні нагороди
- Обмежені skіни та аватари за сезон

### 2.6 Pet / Компаньйон
- Віртуальний персонаж, що росте разом із заощадженнями
- Різні стадії еволюції
- Кастомізація зовнішності за кристали

## 3. Фінансові інструменти

### 3.1 Бюджетний трекер
- Категорії витрат: їжа, транспорт, розваги, одяг тощо
- Кругові діаграми за тиждень / місяць
- Автоматична класифікація транзакцій

### 3.2 Цілі заощаджень
- Візуальний прогрес-бар до конкретної мети
- Приклади: "На ноутбук — 25 000 грн", "На відпустку — 15 000 грн"
- Таймер + підказка "скільки щодня треба відкладати"

### 3.3 Підключення реальних банків
- Монобанк, ПриватБанк та інші через API
- Автоматична синхронізація транзакцій
- Категоризація по фактичним витратам

### 3.4 Автоматичні перекази
- Перекази за розкладом (щотижня, щомісяця)
- Правило "округлення": 47 грн → 50 грн, 3 грн у скарбничку
- Автоматичні "заощадження" без натискань

### 3.5 Міні-інвестиції
- Умовні інвестиції всередині додатка
- "Відкласти в ETF" симуляція
- Навчальний режим для новачків

### 3.6 "Розумні цілі"
- Додаток сам пропонує суму щоденного внеску
- Розрахунок на основі дедлайну та суми цілі
- Динамічна адаптація при пропущених днях

### 3.7 Фінансові поради від AI
- Персоналізовані підказки на основі патернів витрат
- Щотижневий фінансовий дайджест
- Рекомендації щодо оптимізації бюджету

## 4. Візуал та кастомізація

### 4.1 Теми оформлення
- **Neon** — поточна (блакитно-рожева)
- **Gold** — золоті акценти на темному фоні
- **Crimson** — червона палітра
- **Dark Synthwave** — ретро-хвилі, пурпурно-неонові градієнти
- **Matrix** — зелений текст на чорному, цифровий дощ
- **Vaporwave** — пастельні тони, ретро-естетика

### 4.2 Кастомні аватари
- Конструктор з частинами: голова, тіло, неон, ефекти
- Unlockable деталі за рівнем / досягненнями
- Анімовані аватари на сторінці профілю

### 4.3 Анімовані фони
- Залежать від прогресу заощаджень
- Чим більше грошей — тим крутіший кібер-світ
- Particle-ефекти, "кібер-дощ", неонові хвилі

### 4.4 3D-скарбничка
- Анімована 3D-модель, що наповнюється монетами
- Візуальний фідбек за кожен внесок
- Різні стадії: порожня → напівповна → повна

### 4.5 Світла тема
- Перемикач Dark / Light mode
- Зручність вдень / вночі
- Збереження вибору в налаштуваннях

## 5. Мотивація та психологія

### 5.1 AI-коуч
- Штучний інтелект, що пише мотиваційні повідомлення
- Контекстні повідомлення залежно від прогресу
- Підтримка при "слабких" моментах

### 5.2 Стрік-захист
- Якщо пропустив день → можна витратити кристали для збереження стріку
- "Shield" на 1 / 3 / 7 днів
- Баланс між стимулом та стрімкістю

### 5.3 Щоденний журнал настрою
- Реєстрація настрою + сума внеску
- Відстеження кореляції настрою та заощаджень
- Аналітика трендів на місяць

### 5.4 Стрік-система (покращена)
- Візуальний полум'яний стрік (вже є: 1 день)
- Тирири: 7 днів, 30 днів, 100 днів, 365 днів
- Унікальні нагороди за кожний тирий
